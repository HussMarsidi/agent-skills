# Project Context

## Purpose
This project provides a CLI tool (`create-cursor-skill`) to generate Cursor skill templates that comply with the Agent Skills specification. The CLI creates properly structured skill directories with validated names, compliant YAML frontmatter, and placeholder content, streamlining the skill development process.

## Tech Stack
- TypeScript
- Node.js
- Commander.js (CLI framework)
- Zod (schema validation)
- chalk (terminal colors)
- ora (loading spinners)
- fs-extra (file operations)
